Экспоненциальные производящие функции

Иногда производящая функция последовательности ana_nan​ сложна и неудобна для работы, при этом связанная с ней последовательность an/n!a_n/n!an​/n! имеет простую производящую функцию. Чтобы упростить себе жизнь, в таких случаях мы будем работать с an/n!a_n/n!an​/n! и в самом конце выполним умножение на n!n!n!. Делать так приходится часто, поэтому введём отдельное определение.

Экспоненциальная производящая функция

Слово "экспоненциальная" употребляется здесь, так как функция eze^zez будет ЭПФ для последовательности единиц.

Действия, которые мы проделывали с обычными производящими функциями будут давать немного другой результат.

Дзета-функция Римана

Наиболее распространённым частным случаем ПФД является Дзета-функция Римана — ПФД для последовательности единиц. Здесь накладывается условие Re⁡z>1\Re z > 1Rez>1

ζ(z)=∑n=1∞1nz\zeta(z) = \sum\limits_{n=1}^{\oo} \frac{1}{n^z}ζ(z)=n=1∑∞​nz1​

Удивительная связь дзета-функции с простыми числами устанавливается через произведение Эйлера. Чтобы вывести его, снова обратимся к сумме бесконечной геометрической прогрессии:

∑n=0∞xn=1+x+x2+⋯=11−x\sum\limits_{n=0}^{\oo} x^n = 1 + x + x^2 + \dotsb = \frac{1}{1-x}n=0∑∞​xn=1+x+x2+⋯=1−x1​

Применим эту формулу для одного простого числа ppp. Пусть x=p−zx = p^{-z}x=p−z. Тогда сумма обратных степеней этого простого числа равна:

∑n=0∞1(pk)z=1+1pz+1p2z+⋯=11−p−z\sum\limits_{n=0}^\oo \frac{1}{(p^k)^z} = 1 + \frac{1}{p^z} + \frac{1}{p^{2z}} + \dotsb = \frac{1}{1 - p^{-z}}n=0∑∞​(pk)z1​=1+pz1​+p2z1​+⋯=1−p−z1​

Теперь рассмотрим произведение таких выражений по всем простым числам ppp:

∏p11−p−z=(1+12z+14z+⋯ ) (1+13z+19z+⋯ ) (1+15z+125z+⋯ )⋯\prod\limits_p \frac{1}{1 - p^{-z}} = \left( 1 + \frac{1}{2^z} + \frac{1}{4^z} + \dotsb \right) \, \left( 1 + \frac{1}{3^z} + \frac{1}{9^z} + \dotsb \right) \, \left( 1 + \frac{1}{5^z} + \frac{1}{25^z} + \dotsb \right) \cdotsp∏​1−p−z1​=(1+2z1​+4z1​+⋯)(1+3z1​+9z1​+⋯)(1+5z1​+25z1​+⋯)⋯

Раскрывая скобки в этом бесконечном произведении, мы будем перемножать слагаемые вида p−kzp^{-kz}p−kz. В силу основной теоремы арифметики, каждое натуральное число nnn единственным образом представляется в виде произведения простых множителей.

Следовательно, в результате раскрытия скобок мы получим сумму обратных степеней n−zn^{-z}n−z для всех натуральных чисел nnn, причем ровно по одному разу. Таким образом, бесконечное произведение превращается в ряд Дирихле:

∏p(1−p−z)−1=∑n=1∞1nz=ζ(z)\prod\limits_p (1 - p^{-z})^{-1} = \sum\limits_{n=1}^\oo \frac{1}{n^z} = \zeta(z)p∏​(1−p−z)−1=n=1∑∞​nz1​=ζ(z)
