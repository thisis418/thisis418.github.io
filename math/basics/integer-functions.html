Целочисленные функции

Целочисленной называется функция f ⁣:X→Zf \colon X \to \ZZf:X→Z, которая отображает какое-то числовое множество в Z\ZZZ.

Начнем с самых широко используемых функций — пола и потолка

⌊x⌋=наибольшее целое, не превышающее x⌈x⌉=наименьшее целое, не меньшее x\align{ \lfloor x \rfloor &= \text{наибольшее целое, не превышающее}~ x \\ \lceil x \rceil &= \text{наименьшее целое, не меньшее}~ x }⌊x⌋⌈x⌉​=наибольшее целое, не превышающее x=наименьшее целое, не меньшее x​

В целых точках значения функций совпадают: ⌊x⌋=x=⌈x⌉ ⇔ x целое\lfloor x \rfloor = x = \lceil x \rceil ~\Leftrightarrow~ x ~\text{целое}⌊x⌋=x=⌈x⌉ ⇔ x целое, так же ⌊x⌋=⌈x⌉−1 ⇔ x не целое\lfloor x \rfloor = \lceil x \rceil - 1 ~\Leftrightarrow~ x ~\text{не целое}⌊x⌋=⌈x⌉−1 ⇔ x не целое.

Обе функции являются отражением друг друга относительно обеих осей: ⌊−x⌋=−⌈x⌉ и ⌈−x⌉=−⌊x⌋\lfloor -x \rfloor = - \lceil x \rceil ~\text{и}~ \lceil -x \rceil = - \lfloor x \rfloor⌊−x⌋=−⌈x⌉ и ⌈−x⌉=−⌊x⌋.
