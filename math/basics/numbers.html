Комплексные числа

Часто сложные проблемы, возникающие при оперировании целыми числами, решаются действительными. А сложные проблемы, возникающие с действительными числами решаются с помощью более общего класса чисел — комплексными. Это числа zzz, представимые в виде z=x+iyz = x + iyz=x+iy.

C={x+iy ; x,y∈R}\CC = \{x + iy \,;~ x, y \in \RR \}C={x+iy; x,y∈R}

iii — мнимая единица, особое число, для которого верно i2=−1i^2 = -1i2=−1.

Число x=Re⁡zx = \Re zx=Rez называется действительной частью числа zzz, а число y=Im⁡zy = \Im zy=Imz мнимой частью.

Теперь определим величину, называемую модулем комплексного числа z=x+iyz = x + iyz=x+iy как

∣z∣=x2+y2=(Re⁡z)2+(Im⁡z)2|z| = \sqrt{x^2 + y^2} = \sqrt{\bigl(\Re z\bigr)^2 + \bigl(\Im z\bigr)^2}∣z∣=x2+y2​=(Rez)2+(Imz)2​
