Pack bits

Программисты из Apple придумали другой алгоритм, дающий стабильно неплохие результаты.

Давайте схлопывать последовательности, состоящие из одного повторяющегося символа. Выделим в исходной строке символы, которые повторяются более двух раз и вычислим счётчики

"aaaaaldaaarttttoxevvv" -> ("a" * 5) + "ld" + ("a" * 3) + "r" + ("t" * 4) + "oxe" + ("v" * 3)

У нас получилась последовательность блоков двух типов: сжатый и несжатый. Сжатые блоки содержат один символ и количество повторений этого символа. Несжатые блоки содержат сами несжатые данные.
