Абстрактные таблицы

Большие наборы записей и скалярных типах нужно как-то хранить.

Записью всегда выступает тип (возможно, комплексный), описание которого мы заранее знаем полностью. Например, если мы делаем сервис для доставки еды, и нам нужно хранить пользователей, то для них мы сделаем запись

record User: string name int age string address
