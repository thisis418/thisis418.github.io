Сочетания

Сочетания из nnn объектов по kkk — все возможные варианты выбора kkk различных элементов из набора nnn объектов без учёта порядка.

Например, сочетания из 555 объектов {a,b,c,d,e}\{a, b, c, d, e\}{a,b,c,d,e} по 333 это

abc,abd,abe,acd,ace,ade,bcd,bce,bde,cdeabc,\quad abd,\quad abe,\quad acd,\quad ace,\quad ade,\quad bcd,\quad bce,\quad bde,\quad cdeabc,abd,abe,acd,ace,ade,bcd,bce,bde,cde

Количество сочетаний из nnn по kkk ещё называется биномиальным коэффициентом и обозначается

(nk)   ⁣=def   ⁣n!(n−k)!⋅k!\binom{n}{k} \defeq \frac{n!}{(n-k)! \cdot k!}(kn​)=def(n−k)!⋅k!n!​

В нашем примере

(53)=5!(5−3)!⋅3!=5⋅4⋅33⋅2⋅1=10\binom{5}{3} = \frac{5!}{(5-3)! \cdot 3!} = \frac{5 \cdot 4 \cdot 3}{3 \cdot 2 \cdot 1} = 10(35​)=(5−3)!⋅3!5!​=3⋅2⋅15⋅4⋅3​=10

Мы раскрываем скобки в выражении

(x+y)n=(x+y) (x+y) (x+y) ⋯ (x+y)(x+y)^n = (x+y) \, (x+y) \, (x+y) \, \dotsm \, (x+y)(x+y)n=(x+y)(x+y)(x+y)⋯(x+y)

Давайте разберемся, как появляется конкретный член xk yn−kx^k \, y^{n-k}xkyn−k. Чтобы получился такой член, нам нужно ровно kkk раз выбрать xxx и, соответственно, n−kn-kn−k раз выбрать yyy из наших nnn скобок.

Этот коэффициент по определению является биномиальным коэффициентом (nk)\binom{n}{k}(kn​).

Вещественное обобщение

Формулу из определения биномиальных коэффициентов можно обобщить и на случай нецелых nnn

(rk)   ⁣=def   ⁣{rk‾k!=∏i=1kr+1−iik⩾00k<0\binom{r}{k} \defeq \cases{ \displaystyle \frac{r^{\underline{k}}}{k!} = \prod\limits_{i=1}^k \frac{r+1-i}{i} & k \ge 0 \\ 0 & k < 0 }(kr​)=def⎩⎨⎧​k!rk​​=i=1∏k​ir+1−i​0​k⩾0k<0​

При этом, для определенности, при k<0k < 0k<0 полагаем (rk)=0\binom{r}{k} = 0(kr​)=0.

Различные свойства и формулы применимы при разных условиях на переменные. Далее я буду писать переменные nnn и kkk, подразумевая, что они целые, а rrr у меня будет действительной переменной.

Свойства биномиальных коэффициентов

Симметрия. Из определения получаем, что биномиальные коэффициенты симметричны в том смысле, что

(nk)=(nn−k)\binom{n}{k} = \binom{n}{n-k}(kn​)=(n−kn​)

Вынесение и внесение. Опять же, из определения можно вынести по одному члену и получить формулу

(rk)=rk⋅(r−1k−1)\binom{r}{k} = \frac{r}{k} \cdot \binom{r-1}{k-1}(kr​)=kr​⋅(k−1r−1​)

Используя это свойство, можно получить два очень полезных факта, которые используются повсеместно

k⋅(rk)=r⋅(r−1k−1)и1r⋅(rk)=1k⋅(r−1k−1)k \cdot \binom{r}{k} = r \cdot \binom{r-1}{k-1} \quad\text{и}\quad \frac{1}{r} \cdot \binom{r}{k} = \frac{1}{k} \cdot \binom{r-1}{k-1}k⋅(kr​)=r⋅(k−1r−1​)иr1​⋅(kr​)=k1​⋅(k−1r−1​)

Применяя поочередно свойства внесения-вынесения и симметрии, можно получить еще одну формулу

r⋅(r−1k)=(r−k)⋅(rk)r \cdot \binom{r-1}{k} = (r-k) \cdot \binom{r}{k}r⋅(kr−1​)=(r−k)⋅(kr​)

Формула сложения. Используя несколько предыдущих свойств получаем одно из самых важных тождеств

(rk)=(r−1k)+(r−1k−1)\binom{r}{k} = \binom{r-1}{k} + \binom{r-1}{k-1}(kr​)=(kr−1​)+(k−1r−1​)

Получившаяся формула является рекуррентным соотношением для биномиальных коэффициентов, то есть с её помощью мы можем вычислять биномиальные коэффициенты методами динамического программирования.

Также эта формула показывает связь биномиальных коэффициентов с числами в треугольнике Паскаля. Элемент под номером kkk в строке nnn равен сумме его «родителей» под номерами kkk и k−1k-1k−1 из строки n−1n-1n−1.

Формула суммирования по обеим индексам. Из формулы сложения можно рекуррентно вывести ещё одну полезную формулу, если раскладывать биномиальные коэффициенты так, чтобы убывал нижний индекс.

(r+n+1n)=∑k=0n(r+kk)=(r0)+(r+11)+(r+22)+⋯+(r+n−1n−1)+(r+nn)\binom{r+n+1}{n} = \sum\limits_{k=0}^n \binom{r+k}{k} = \binom{r}{0} + \binom{r+1}{1} + \binom{r+2}{2} + \dotsb + \binom{r+n-1}{n-1} + \binom{r+n}{n}(nr+n+1​)=k=0∑n​(kr+k​)=(0r​)+(1r+1​)+(2r+2​)+⋯+(n−1r+n−1​)+(nr+n​)

Формула суммирования по верхнему индексу. Предыдущая формула выражает один биномиальный коэффициент в виде суммы других, верхний и нижний индексы которых остаются «равноудалёнными». Если раскладывать по-другому, чтобы нижний индекс сохранялся, то мы получим другую, не менее полезную формулу.

(n+1m+1)=∑k=0n(km)=(0m)+(1m)+(2m)+⋯+(n−1m)+(nm)\binom{n+1}{m + 1} = \sum\limits_{k=0}^n \binom{k}{m} = \binom{0}{m} + \binom{1}{m} + \binom{2}{m} + \dotsb + \binom{n-1}{m} + \binom{n}{m}(m+1n+1​)=k=0∑n​(mk​)=(m0​)+(m1​)+(m2​)+⋯+(mn−1​)+(mn​)

Формула вернего обращения. Она описывает связь между положительными отрицательными верхними индексами. Эта формула следует из определения (rk)\binom{r}{k}(kr​), в котором каждый верхний множитель домножен на −1-1−1.

(rk)=(−1)k⋅(k−r−1k)\binom{r}{k}=(-1)^k \cdot \binom{k-r-1}{k}(kr​)=(−1)k⋅(kk−r−1​)

Мультиномиальные коэффициенты

Рассматривая разложение (x+y)n(x+y)^n(x+y)n мы получали в качестве коэффициентов при члене xk yn−kx^k \, y^{n-k}xkyn−k биномиальный коэффициент (nk)\binom{n}{k}(kn​). Давайте рассмотрим степень не биноме, а полинома с mmm переменными (x1+x2+⋯+xm)n(x_1 + x_2 + \dotsb + x_m)^n(x1​+x2​+⋯+xm​)n. Какие коэффициенты мы получим?

Мы получим мультиномиальные коэффициенты, которые, по аналогии с биномиальными, выражают количество способов выбрать из nnn предметов ровно k1k_1k1​ предмет типа 111, k2k_2k2​ предметов типа 222, ..., kmk_mkm​ предметов типа mmm.

Для n=k1+k2+…+kmn = k_1 + k_2 + \dotsc + k_mn=k1​+k2​+…+km​ мультиномиальный коэффициент порядка mmm

Свёртка Вандермонда

Это тождество выполняемое для r,s∈Rr,s \in \RRr,s∈R и n∈Zn \in \ZZn∈Z

∑k(rk)(sn−k)=(r+sn)\sum\limits_{k} \binom{r}{k}\binom{s}{n-k} = \binom{r+s}{n}k∑​(kr​)(n−ks​)=(nr+s​)

Её можно вывести из комбинаторной интерпритации. Пусть rrr и sss - целые неотрицательные числа. (Общий случай получется посредством полиномиальной аргументации). В правой части (r+sn)\binom{r+s}{n}(nr+s​) - это число способов выбрать nnn человек из rrr мужчин и sss. В левой части каждый член суммы - это число способов выбрать kkk мужчин и n−kn-kn−k человек из sss женщин. А суммирование по всем kkk учитывает каждую возможность ровно по одному разу.
