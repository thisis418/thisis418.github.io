Булевы функции

Булева функция от nnn переменных — функция f ⁣:{0,1}n→0,1f \colon \{0, 1\}^n \to {0, 1}f:{0,1}n→0,1.

Она принимает на вход nnn битов x1,x2,…,xnx_1, x_2, \dotsc, x_nx1​,x2​,…,xn​, и возвращает один бит f(x1,x2,…,xn)f(x_1, x_2, \dotsc, x_n)f(x1​,x2​,…,xn​).

Каждая булева функция от nnn переменных может быть определена своей таблицей истинности, состоящей из значений этой функции на всех 2n2^n2n входах. Часто входы сортируют в лексикографическом порядке, и таблицу истинности пишут просто строкой

f(0,0,…,0,0)f(0,0,…,0,1)f(0,0,…,1,0)f(0,0,…,1,1)⋯f(1,1,…,1,0)f(1,1,…,1,1)f(0, 0, \dotsc, 0, 0) \quad f(0, 0, \dotsc, 0, 1) \quad f(0, 0, \dotsc, 1, 0) \quad f(0, 0, \dotsc, 1, 1) \quad \cdots \quad f(1, 1, \dotsc, 1, 0) \quad f(1, 1, \dotsc, 1, 1)f(0,0,…,0,0)f(0,0,…,0,1)f(0,0,…,1,0)f(0,0,…,1,1)⋯f(1,1,…,1,0)f(1,1,…,1,1)

Все булевы функции можно выразить через бинарные операторы, изученные нами в разделе Основы алгебры логики.

Многочлены Жегалкина

Жегалкин заметил, что любую булеву функцию можно представить в виде

f(x1,x2,…,xn−1,xn)=g(x1,x2,…,xn−1)⊕h(x1,x2,…,xn−1)∧xnf(x_1, x_2, \dotsc, x_{n-1}, x_n) = g(x_1, x_2, \dotsc, x_{n-1}) \oplus h(x_1, x_2, \dotsc, x_{n-1}) \land x_nf(x1​,x2​,…,xn−1​,xn​)=g(x1​,x2​,…,xn−1​)⊕h(x1​,x2​,…,xn−1​)∧xn​

где g(x1,x2,…,xn−1)=f(x1,x2,…,xn−1,0)g(x_1, x_2, \dotsc, x_{n-1}) = f(x_1, x_2, \dotsc, x_{n-1}, 0)g(x1​,x2​,…,xn−1​)=f(x1​,x2​,…,xn−1​,0) и h(x1,x2,…,xn−1)=f(x1,x2,…,xn−1,0)⊕f(x1,x2,…,xn−1,1)h(x_1, x_2, \dotsc, x_{n-1}) = f(x_1, x_2, \dotsc, x_{n-1}, 0) \oplus f(x_1, x_2, \dotsc, x_{n-1}, 1)h(x1​,x2​,…,xn−1​)=f(x1​,x2​,…,xn−1​,0)⊕f(x1​,x2​,…,xn−1​,1).

Повторяя эту операцию итеративно для всех внутренних функций, можно получить разложение функции fff в многочлен, называемый многочленом Жегалкина

f(x1,x2,…,xn−1,xn)=a0⊕⨁1⩽k⩽n1⩽j1<⋯<jk⩽naj1,…,jk∧⋀s=1kxjsf(x_1, x_2, \dotsc, x_{n-1}, x_n) = a_0 \oplus \bigoplus_{\mathclap{\substack{1 \;\! \le \;\! k \le n \\ 1 \le j_1 < \dotsb < j_k \le n}}} a_{j_1, \dotsc, j_k} \land \bigland_{s=1}^k x_{j_s}f(x1​,x2​,…,xn−1​,xn​)=a0​⊕1⩽k⩽n1⩽j1​<⋯<jk​⩽n​⨁​aj1​,…,jk​​∧s=1⋀k​xjs​​

Операция ⊕\oplus⊕ аналогична обычному сложению по модулю 222, а операция ∧\land∧ аналогична умножению, значит, многочлен Жегалкина можно переписать из булевой вида в вид многочлена из Z2[x1,x2,…,xn]\ZZ_2[x_1, x_2, \dotsc, x_n]Z2​[x1​,x2​,…,xn​]

f(x1,x2,…,xn−1,xn)=(a0+∑1⩽k⩽n1⩽j1<⋯<jk⩽naj1,…,jk⋅∏s=1kxjs) mod 2f(x_1, x_2, \dotsc, x_{n-1}, x_n) = \Biggl( a_0 + \sum\limits_{\mathclap{\substack{1 \;\! \le \;\! k \le n \\ 1 \le j_1 < \dotsb < j_k \le n}}} a_{j_1, \dotsc, j_k} \cdot \prod\limits_{s=1}^k x_{j_s} \Biggr) \bmod 2f(x1​,x2​,…,xn−1​,xn​)=(a0​+1⩽k⩽n1⩽j1​<⋯<jk​⩽n​∑​aj1​,…,jk​​⋅s=1∏k​xjs​​)mod2

Такой вид функции f(x1,x2,…,xn)f(x_1, x_2, \dotsc, x_n)f(x1​,x2​,…,xn​) ещё называется её мультилинейным представлением. Коэффициенты aj1,…,jk   ⁣∈   ⁣{0,1}a_{j_1, \dotsc, j_k} \;\! \in \;\! \{0, 1\}aj1​,…,jk​​∈{0,1} показывают, содержится ли член ∏s=1kxjs\prod\limits_{s=1}^k x_{j_s}s=1∏k​xjs​​ в представлении функции, или нет.

Нормальные формы

Джордж Буль разлагал функции по-другому

f(x1,x2,…,xn−1,xn)=(g(x1,x2,…,xn−1)∧xn)∨(h(x1,x2,…,xn−1)∧xˉn)f(x_1, x_2, \dotsc, x_{n-1}, x_n) = \bigr( g(x_1, x_2, \dotsc, x_{n-1}) \land x_n \bigr) \lor \bigl( h(x_1, x_2, \dotsc, x_{n-1}) \land \bar{x}_n \big)f(x1​,x2​,…,xn−1​,xn​)=(g(x1​,x2​,…,xn−1​)∧xn​)∨(h(x1​,x2​,…,xn−1​)∧xˉn​)

где просто g(x1,x2,…,xn−1)=f(x1,x2,…,xn−1,0)g(x_1, x_2, \dotsc, x_{n-1}) = f(x_1, x_2, \dotsc, x_{n-1}, 0)g(x1​,x2​,…,xn−1​)=f(x1​,x2​,…,xn−1​,0) и h(x1,x2,…,xn−1)=f(x1,x2,…,xn−1,1)h(x_1, x_2, \dotsc, x_{n-1}) = f(x_1, x_2, \dotsc, x_{n-1}, 1)h(x1​,x2​,…,xn−1​)=f(x1​,x2​,…,xn−1​,1).

Аналогично, итеративно раскрывая внутренние функции, можно получить представление исходной функции в дизъюнктивной нормальной форме
