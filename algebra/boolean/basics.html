Основы алгебры логики

Алгебра логики представляет собой алгебру над высказываниями. Нас интересует только истинность или ложность высказываний.

Бинарные операции

Имеется 161616 функций f ⁣:{0,1}2→{0,1}f \colon \{0, 1\}^2 \to \{0, 1\}f:{0,1}2→{0,1}, переводящих два бита xxx и yyy в один бит f(x,y)f(x, y)f(x,y). Такие функции, следуя терминологии алгебры, называются бинарными операциями над битами.

Функция от двух переменных часто записывается в суффиксной нотации, то есть в виде оператора x∗yx * yx∗y, а не f(x,y)f(x, y)f(x,y). Столбец «Символ оператора» как раз показывает символ, который нужно вставить вместо ∗*∗ для записи разных операций.

И, ИЛИ и НЕ

Обе операции ∧\land∧ и ∨\lor∨ являются коммутативными и ассоциативными:

x∧y=y∧xи(x∧y)∧z=x∧(y∧z)x∨y=y∨xи(x∨y)∨z=x∨(y∨z)x \land y = y \land x \quad\text{и}\quad (x \land y) \land z = x \land (y \land z)\\[0.4em]x \lor y = y \lor x \quad\text{и}\quad (x \lor y) \lor z = x \lor (y \lor z) x∧y=y∧xи(x∧y)∧z=x∧(y∧z)x∨y=y∨xи(x∨y)∨z=x∨(y∨z)

А ещё ∧\land∧ дистрибутивна по ∨\lor∨, и наоборот, ∨\lor∨ дистрибутивна по ∧\land∧:

x∧(y∨x)=(x∧y)∨(x∧z)иx∨(y∧z)=(x∨y)∧(x∨z)x \land (y \lor x) = (x \land y) \lor (x \land z) \quad\text{и}\quad x \lor (y \land z) = (x \lor y) \land (x \lor z)x∧(y∨x)=(x∧y)∨(x∧z)иx∨(y∧z)=(x∨y)∧(x∨z)

Также работает идемпотентность

x∨x=xиx∧x=xx \lor x = x \quad\text{и}\quad x \land x = xx∨x=xиx∧x=x

и поглощение

x∧(x∨y)=xиx∨(x∧y)=xx \land (x \lor y) = x \quad\text{и}\quad x \lor (x \land y) = xx∧(x∨y)=xиx∨(x∧y)=x
