Быстрое возведение в степень

Пусть у нас есть какая-то ассоциативная бинарная операция ∘\circ∘ в алгебре A\AAAA. Мы хотим научиться быстро считать степень элемента aaa носителя алгебры A\AAAA по этой операции, то есть значение

an=a∘a∘⋯∘a∘a⏟n разa^n = \underbrace{a \circ a \circ \dotsb \circ a \circ a}_{n ~\text{раз}}an=n разa∘a∘⋯∘a∘a​​

В псевдокоде я заменю операцию ∘\circ∘ на умножение, но это просто для удобства написания. В формулах я буду использовать оригинальное обозначение, дабы подчеркнуть универсальность.

Можно применить свойство ассоциативности для сведения задачи возведения в степень nnn к вдвое меньшей задаче, используя простое свойство

an={an/2∘ann чётноеa∘an−1n нечётноеa^n = \cases{a^{n/2} \circ a^n & n~\text{чётное} \\ a \circ a^{n-1} & n~\text{нечётное}}an={an/2∘ana∘an−1​n чётноеn нечётное​

Из этого простого свойства можно сразу вывести простой рекурсивный алгоритм

function fast_pow(a, int n): if n % 2 == 0: return fast_pow(a, n/2) ** 2 else: return a * fast_pow(a, n-1)

У рекурсии есть оверхед, от которого хотелось бы избавиться.

Давайте посмотрим на двоичное представление nnn

function fast_pow(a, int n): int result = 1 while n > 0: if n % 2 == 1: result *= a a *= a n /= 2; return result

Оценим количество умножений, требуемых алгоритмом (не важно каким, они одинаковые). Пусть для возведения числа aaa в степень nnn требуется M(n)M(n)M(n) умножений. Тогда, исходя из нашего алгоритма, справедлива формула

M(n)=M(⌊n2⌋)+1+(n mod 2)M(n) = M\left( \left\lfloor \frac{n}{2} \right\rfloor \right) + 1 + (n \bmod 2)M(n)=M(⌊2n​⌋)+1+(nmod2)

Решение этого уравнения

M(n)=⌊log⁡2n⌋+ν(n)−1M(n) = \lfloor \log_2 n \rfloor + \nu(n) - 1M(n)=⌊log2​n⌋+ν(n)−1

где ν(n)\nu(n)ν(n) — количество единиц в числе nnn
