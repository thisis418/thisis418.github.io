Индуктивные функции

Напомню, что X∗X^*X∗ — множество конечных кортежей из элементов множества XXX, и ω↼x\omega \concat xω↼x — операция присоединения элемента x∈Xx \in Xx∈X в конец кортежа ω∈X∗\omega \in X^*ω∈X∗.

Индуктивная функция

Функция f ⁣:X∗→Af \colon X^* \to Af:X∗→A называется индуктивной, если существует такая функция F ⁣:X∗×X→AF \colon X^* \times X \to AF:X∗×X→A, что

f(ω↼x)=F(f(ω),x)f(\omega \concat x) = F \big(f(\omega), x\big)f(ω↼x)=F(f(ω),x)

Функция FFF называется функцией пересчёта функции fff.

Другими словами, мы можем восстановить значение функции fff на кортеже

f(x1,x2,…,xn−1,xn)=F(f(x1,x2,…,xn−1),xn)f(x_1, x_2, \dotsc, x_{n-1}, x_n) = F \big( f(x_1, x_2, \dotsc, x_{n-1}), x_n \big)f(x1​,x2​,…,xn−1​,xn​)=F(f(x1​,x2​,…,xn−1​),xn​)

Значение индуктивной функции для любого кортежа можно легко найти, если известна функция пересчёта

f(x1,x2,…,xn−1,xn)=F(F(F(…F(F(f(ε),x1),x2),… ),xn−1),xn)f(x_1, x_2, \dotsc, x_{n-1}, x_n) = F (F(F(\dots F(F(f(\varepsilon), x_1), x_2), \dots), x_{n-1}), x_n)f(x1​,x2​,…,xn−1​,xn​)=F(F(F(…F(F(f(ε),x1​),x2​),…),xn−1​),xn​)

Соответственно, код, вычисляющий значение fff

function f(tuple x): result = f(empty) for element in x: result = F(result, element) return result

Для кортежа длины nnn время работы алгоритма составляет O(n)⋅time⁡FO(n) \cdot \time FO(n)⋅timeF, где time⁡F\time FtimeF — время работы функции FFF.

Стационарные значения

В некоторых случаях можно сократить вычисление индуктивной функции. Если в процессе вычисления мы получили значение, которое никакими перевычислениями не изменить, то функцию можно дальше не считать и вернуть результат. Такие значения для функции называются стационарными.

Стационарное значение индуктивной функции fff — такое значение yyy, что

∀ x∈XF(y,x)=y\forall\, x \in X \? F(y, x) = y∀x∈XF(y,x)=y

Или, по-другому,

∀ x∈Xf(ω↼x)=f(ω)\forall\, x \in X \? f(\omega \concat x) = f(\omega)∀x∈Xf(ω↼x)=f(ω)

То есть, если мы достигли стационарного значения, можно сразу же объявить ответ, не досчитывая до конца.

С учетом этого, код вычисления индуктивной функции немного изменится

function f(tuple x): result = f(empty) for element in x: result = F(result, element) break if result is стационарное значение f() return result
