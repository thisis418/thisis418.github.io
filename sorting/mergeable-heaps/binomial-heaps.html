Биномиальное дерево

Биномиальное дерево BrB_rBr​ ранга rrr — дерево, определяемое рекурсивно: B0B_0B0​ — дерево, состоящее просто из одного узла. BrB_rBr​ состоит из двух биномиальных деревьев ранга r−1r-1r−1, при корень одного из этих деревьев становится крайним левым ребёнком другого.

Br={{⋅} if r=0Br−1⊔{Br−1} if r   ⁣⩾   ⁣1B_r = \cases{ \{\cdot\} & \if r = 0 \\ B_{r-1} \djunion \{ B_{r-1} \} & \if r \;\! \ge \;\! 1 }Br​={{⋅}Br−1​⊔{Br−1​}​ if r=0 if r⩾1​

Можно раскрыть рекурсию и получить эквивалентное определение. BrB_rBr​ состоит из корня, у которого есть ровно rrr детей — корни поддеревьев B0,B1,…,Br−1B_0, B_1, \dotsc, B_{r-1}B0​,B1​,…,Br−1​, расположенных в порядке убывания рангов.

Br={{⋅} if r=0{B0}⊔{B1}⊔⋯⊔{Br−1} if r   ⁣⩾   ⁣1B_r = \cases{ \{\cdot\} & \if r = 0 \\ \{ B_0 \} \djunion \{ B_1 \} \djunion \dotsb \djunion \{ B_{r-1} \} & \if r \;\! \ge \;\! 1 }Br​={{⋅}{B0​}⊔{B1​}⊔⋯⊔{Br−1​}​ if r=0 if r⩾1​

Отмечу сразу несколько очевидных свойств биномиальных деревьев.

В биномиальном дереве ранга rrr содержится ровно 2r2^r2r вершин. Отсюда следует, что если в биномиальном дереве содержится nnn вершин, то оно имеет ранг log⁡2n\log_2 nlog2​n.

Биномиальное дерево ранга rrr имеет высоту r+1r+1r+1, при этом на глубине kkk содержится (rk)\binom{r}{k}(kr​) вершин. Корень такого дерева имеет степень rrr, а все потомки имеют степень меньше rrr. То есть если в биномиальном дереве содержится nnn элементов, то максимальная степень произвольного узла равна log⁡2n\log_2 nlog2​n.

Биномиальная куча

Биномиальная куча — лес из биномиальных деревьев различных рангов. Каждое биномиальное дерево в этом лесу удовлетворяет свойству кучи: приоритет любого узла больше приоритетов его детей.

Поскольку технически детей у каждого узла может быть много, создавать статическую структуру не получится. Давайте у каждого узла всех детей хранить в односвязном списке, тогда в самом узле нужно будет хранить только ссылку на самого левого ребёнка. Поскольку поддеревья у каждого корня упорядочены по убыванию ранга, порядок детей всегда фиксированный, и хранить детей в такой структуре можно.

Обязательно для каждого узла нужно хранить ранг поддерева с корнем в этом узле. Учтём, что ранг узла совпадает с его степенью.
